# What kind of problems skill fixes
- Route issues - Flask routes not working, getting a 404 error, Blueprint connection problems
- Template issues - missing variables, Jinja2 not working
- Database issues - SQLAlchemy errors, connection failures
- Auth issues - login not working, password validation failing, session problems
- Frontend issues - JS not working, CSS not applying, form submission fails

# How the process should look like
Firstly automatically check logs, read relevant files and diagnose the issue. If that did not help: run the app, try to reproduce and repor findings.

# How to explain and fix error 
Firstly explain why this error happenned and the way how you plan to fix this. After this with my permission fix this problem. After fixing explain what was changed and why
