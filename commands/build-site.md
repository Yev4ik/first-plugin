---
name: build-site
description: Steps to perform before building a new web application based on JS, HTML+CSS, Python and SQLite. Use when user says "build web application" "create simple web app" "generate web application" "i want web application"
---

Ask the user the following questions before building:
1. What is your website about?
2. Which pages do you need? (home, about, contact...)
3. Do you need a database? If yes, what data to store
4. Any design preferences? (dark/light theme, colors, style)

After collecting answers, use the skills/build-website skill to create the project.
